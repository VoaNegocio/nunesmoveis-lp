# Imagens do Carrossel

Coloque aqui as imagens dos ambientes premium para o carrossel.

## Estrutura de arquivos:

- `img1.jpg` - Imagem de cozinha planejada (formato 9:16 - vertical)
- `img2.jpg` - Imagem de sala planejada (formato 9:16 - vertical)
- `img3.jpg` - Imagem de closet planejado (formato 9:16 - vertical)
- `img4.jpg` - Imagem de área gourmet (formato 9:16 - vertical)

## Especificações recomendadas:

- **Formato:** JPG ou PNG
- **Proporção:** 9:16 (vertical - fotos tiradas na vertical)
- **Tamanho:** Máximo 2MB por imagem (otimizadas para web)
- **Qualidade:** Alta resolução, iluminação quente, estética premium
- **Orientação:** Vertical (portrait)

## Como adicionar:

1. Coloque as imagens nesta pasta (`public/carrossel/`)
2. Use os nomes exatos: `img1.jpg`, `img2.jpg`, `img3.jpg`, `img4.jpg`
3. As imagens devem ser verticais (9:16) - fotos tiradas na vertical
4. Ou atualize os caminhos no arquivo `src/App.jsx` se usar nomes diferentes
